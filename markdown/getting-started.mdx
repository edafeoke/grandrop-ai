# Get Started

Follow this guide to create your first chatbot and understand the basics of our platform.

## 1. Create a New Chatbot

Click the "New Chatbot" button in the dashboard to start creating your first chatbot. You'll need to:

- Choose a name for your chatbot
- Select a template or start from scratch
- Configure basic settings

```tsx
// Example configuration
const config = {
  name: "Support Bot",
  template: "customer-service",
  settings: {
    language: "en",
    timezone: "UTC",
    responseTime: "instant"
  }
}
